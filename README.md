A desktop application built with electron for generating 128-character alphanumeric pre-shared keys. This is my first project with electron, so mainly for testing the build workflow.

## Download

Download the latest Windows executable from the [Releases](https://github.com/Nat-As/psk_generator_128/releases/latest) page.

## Features
Generate 128-character PSK using only Alphanumeric characters (A-Z, a-z, 0-9)

## Development

```bash
npm install
npm start
```

## Building

```bash
npm run build:win
```

Or use CI/CD workflow with Github actions.
